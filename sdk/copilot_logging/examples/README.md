# Copilot Logging Examples

This directory contains example scripts demonstrating the usage of the `copilot_logging` module.

## Running the Examples

Make sure you have installed the `copilot_logging` package first:

```bash
cd sdk/copilot_logging
pip install -e .
```

Then run the example:

```bash
python examples/example.py
```

## Available Examples

- **example.py**: Comprehensive demonstration of all logging features including:
  - Basic stdout logging with different log levels
  - Silent logger for testing
  - Environment-based configuration
  - Structured logging with rich context
